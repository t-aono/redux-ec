# Redux-EC

`React` - `Redux` を使って EC サイトを構築

## 実装した機能

- ログイン、ログアウト
- 商品情報の管理機能
- カート機能、お気に入り機能
- 注文履歴の管理機能

## バックエンドは Firebase を利用

- Firestore
- Storage
- Hosting

## ローカルサーバー起動

`npm start`

## デプロイ

`npm run build`  
`firebase deploy --only hosting,storage,firestore`  
Functions は無料枠だとデプロイできないのでそれ以外を指定する
