# Redux-EC

## Firebase を利用

- Firestore
- Storage
- Hosting
- Functions

## デプロイ

`npm run build`  
`firebase deploy --only hosting,storage,firestore`  
Functions は無料枠だとデプロイできないのでそれ以外を指定する

## ローカルサーバー起動

`npm start`
